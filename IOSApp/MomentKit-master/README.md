# MomentKit

仿WeChat朋友圈的界面效果，代码整洁易读，静态页面，无网络交互。里面的功能并没有全部完善，可以根据自己的需求去补充一下。欢迎[留言](https://github.com/CheeryLau/MomentKit/issues)，大家一起优化完善。
